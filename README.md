## üì¶ Installation

### CocoaPods

To integrate Instabug into your Xcode project using [CocoaPods](https://cocoapods.org), add it to your `Podfile`:

```ruby
pod 'BugReporting'
```

Then, run the following command:

```bash
$ pod install
```
